# Haltroy.com
This directory contains the source code of the website.

It only requires these:
 - Mobirise 5.3 or newer
 - DeltaPi Site Settings Extension (for Mobirise)
 - Witsec White label Extension (for Mobirise)

These and more are bundled inside the [extensions.zip](https://haltroy.com/extensions.zip) file.
