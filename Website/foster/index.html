Foster

A simple package distribution library for your project.
For C# projects For everything
NuGet Download

Foster (originally HTUPDATE) is a library that allows C# projects to update a certain folder (this can be the project's binary folder) by either installing a Foster package, updating the folder to latest available or install a certain version.

Foster can be used for anything but it requires a C# program to do stuff.

Foster is developed in C# as .NET Standard 2.0 library.

Update your stuff easily

Foster can be used for everything, the only thing that requires is C# code to interact with it.

Foster can update your project in a simple and safe way possible, but you also can use Foster to update almost everything. For example; haltroy's web browser Yorot's package distribution uses Foster to update stuff.

No C#? No Problem.

Foster Manager can be used for doing the exact same thing, but in a command line. So you don't actuall have to write C# code to use Foster since Foster Manager takes care of that part for you.

Foster Manager does not require a C# code to run since it is already compiled software that runs on command-line instead of beign a library.

Foster Manager is developed in C# as .NET Core 3.1 Console App project and contains the Foster's all features with a little extra libraries such as:

  • GZip and Deflate Compression Support (alongside with the default No Compression Mode)
  • AES Encryption Support
  • XML and JSON Parser Support (alongside with the default Binary Format Parser)

Foster Manager is a cross-platform project and available to every system that is supported by .NET Core 3.1.

Here's a list of arguments and what they do:

  • -v | --verbose : Verboses more information.
  • -n | --no-logo : Hides the copyright notice on start.

    • Arguments above can be used with the arguments below.

  • help : Displays the help.
  • info : Displays a certain information about Foster Manager. This can include legal notes or current system configuration.
  • pack : Packs a folder.
  • unpack : Unpacks a package.
  • delta : Creates a delta package from 2 folders with similar content.
  • adelta : Applies a delta package to a folder.
  • create : Batch creates all the packages from a folder containing Foster and all versions to another folder.
  • install : Installs either the latest version or a certain version from a Foster to a folder.
  • update : Installs the latest version from a Foster to a folder to update.

How to use Foster

Foster requires a Foster file stored in web. To create a Foster file, you can follow the Example files.

Foster requires Foster packages to operate. To create them, you can use Foster Manager.

Foster can be initialized with different configurations. Mostly, you can use the Sync() to tell Foster to fetch a Foster file from Web or let it do that job in a periodic way by calling StartSync() function. The auto-sync can also be stopped with StopAutoSync() but it won't take effect until the next check. The time between each sync can be adjusted with SyncInterval property. The sync can be asyncronous if DoTaskAsAsync is true, but the auto-sync is always asynchronous task.

To update to a version, you can call Update() (which can be an asynchronous task) and Foster can do everything on it's own.

You can get what will Foster download and install into the folder with GetDownloads() function.

You can get what is foster doing ath the time by attaching an event to OnLogEntry event.

Distribution notes for Foster Manager

If you are a package maintainer or repository maintainer, you can simply edit Foster Manager's code to fit your operating system, your repository and your packages. Be sure to check notes inside the C# file before building it with the Python script.

If you want to publish a build of Foster Manager, please include links to the GitHub page as upstream link.

Usage of Foster Manager is allowed within the bounds of free software, meaning that Foster or Foster Manager cannot be sold and needs to be free. If you bought either of them, please report it to haltroy and ask for a refund from where you bought it.

Distribution of both Foster and Foster Manager is free of charge.
