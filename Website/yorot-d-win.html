Thank you for downloading Yorot.

Your download should begin shortly. If it doesn't, click here.

Steps

Here are the steps that you should take while installing Yorot to a Windows machine.

1

Confirm

Confirm your download by opening PowerShell on the directory of your download and entering this command:
Get-FileHash Yorot-Installer.exe
The output should be this code:
[[NO VERSION AVAILABLE]]
2

Run the Installer

Run the Installer by simply double clicking on it.
If an antivirus solution warns you about this file, ignore it. It is a false alarm. 
If this happened, feel free to report it to me.
If Windows shows an warning and there's no option for running the installer, just click "Show more info" and "Run anyway".
3

Done!

Just follow the on-screen instructions.
